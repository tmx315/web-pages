# 友情链接申请指南

欢迎[申请友链](/friends/ "申请友链")，一次申请3个哦！

## 申请要求

在申请友链之前，请确保您的网站符合以下条件：

### 基本要求
- 网站能够正常访问
- 网站内容积极向上，无违法违规信息
- 不允许含有病毒、木马、恶意代码、以及任何危害他人计算机安全的内容。
- 网站能够正常访问，加载速度合理
- 包含我方友链

### 申请友链前请先在自己网站添加本站友链，请使用以下信息：
